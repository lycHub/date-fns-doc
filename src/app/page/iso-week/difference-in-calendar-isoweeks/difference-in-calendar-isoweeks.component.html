differenceInCalendarISOWeeks

获取给定日期之间的日历ISO周数。 类型 说明 Number 日历ISO周数 
           // How many calendar ISO weeks are between 6 July 2014 and 21 July 2014?
            var result = differenceInCalendarISOWeeks(
              new Date(2014, 6, 21),
              new Date(2014, 6, 6)
            )
            //=> 3
