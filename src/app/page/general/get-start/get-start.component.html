Introduction

date-fns提供了最全面，最简单和一致的工具集，用于在浏览器和Node.js中操作JavaScript日期。 date-fns就像是日期的 lodash 。它适用于所有场合的140多种功能 
            dateFns.format(new Date(2014, 1, 11), 'MM/DD/YYYY')
            //=> '02/11/2014'

            var dates = [new Date(1995, 6, 2), new Date(1987, 1, 11), new Date(1989, 6, 10)]
            dates.sort(dateFns.compareAsc)
            //=> [
            //   Wed Feb 11 1987 00:00:00,
            //   Mon Jul 10 1989 00:00:00,
            //   Sun Jul 02 1995 00:00:00
            // ]

Installation

该库可作为 npm 包，Bower包提供，也可通过CDN加载。

npm或yarn：

            npm install date-fns --save
            #or
            yarn add date-fns

使用：

          var isToday = require('date-fns/is_today')
          isToday(new Date())
          //=> true

Bower：

            bower install date-fns

将库添加到构建中并通过window.dateFns访问它：

          dateFns.isToday(new Date())
          //=> true
