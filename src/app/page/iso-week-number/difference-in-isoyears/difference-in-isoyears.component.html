differenceInISOYears

获取给定日期之间的完整ISO周编号年份数。 类型 说明 Number 年份数 
              // How many full ISO week-numbering years are between 1 January 2010 and 1 January 2012?
              var result = differenceInISOYears(
                new Date(2012, 0, 1),
                new Date(2010, 0, 1)
              )
              //=> 1
