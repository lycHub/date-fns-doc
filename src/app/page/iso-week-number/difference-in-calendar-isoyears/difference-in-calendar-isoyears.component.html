differenceInCalendarISOYears

获取给定日期之间的日历ISO周编号年数。 类型 说明 Number 年数 
            // How many calendar ISO week-numbering years are 1 January 2010 and 1 January 2012?
            var result = differenceInCalendarISOYears(
              new Date(2012, 0, 1),
              new Date(2010, 0, 1)
            )
            //=> 2
