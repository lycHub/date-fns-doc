isThisWeek

给定日期与当前日期在同一周吗？ 类型 说明 Boolean 给定日期与当前日期是否在同一周 
       // If today is 25 September 2014, is 21 September 2014 in this week?
        var result = isThisWeek(new Date(2014, 8, 21))
        //=> true
