getOverlappingDaysInRanges

获取两个日期范围内重叠的天数。 类型 说明 Number 两个日期范围内重叠的天数。 类型 说明 Error 日期范围的startDate不能在其结束日期之后。 
          // 有3天重叠
          getOverlappingDaysInRanges(
            new Date(2014, 0, 10), new Date(2014, 0, 20), new Date(2014, 0, 17), new Date(2014, 0, 21)
          )
          //=> 3
          
          // 无重叠
          getOverlappingDaysInRanges(
            new Date(2014, 0, 10), new Date(2014, 0, 20), new Date(2014, 0, 21), new Date(2014, 0, 21)
          )
          //=> 0
