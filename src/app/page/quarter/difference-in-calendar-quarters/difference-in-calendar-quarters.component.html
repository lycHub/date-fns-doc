differenceInCalendarQuarters

获取给定日期之间的日历季度数。 类型 说明 Number 日历季度数 
          // How many calendar quarters are between 31 December 2013 and 2 July 2014?
            var result = differenceInCalendarQuarters(
              new Date(2014, 6, 2),
              new Date(2013, 11, 31)
            )
            //=> 3
