compareDesc

比较这两个日期，如果第一个日期在第二个日期之后，返回-1，如果第一个日期在第二个日期之前，返回一个日期，如果日期相等，返回值为0。 类型 说明 Number 比较结果 
           // 比较两个日期
            var result = compareDesc(
              new Date(1987, 1, 11),
              new Date(1989, 6, 10)
            )
            //=> 1
          
           // 日期排序
            var result = [
              new Date(1995, 6, 2),
              new Date(1987, 1, 11),
              new Date(1989, 6, 10)
            ].sort(compareDesc)
            //=> [
            //   Sun Jul 02 1995 00:00:00,
            //   Mon Jul 10 1989 00:00:00,
            //   Wed Feb 11 1987 00:00:00
            // ]
