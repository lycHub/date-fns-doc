closestIndexTo

描述

与给定日期相比，从数组中返回最近日期的索引。

返回

类型 说明 number 从数组中返回最近日期的索引。

注意

类型 说明 TypeError 第二个参数必须是数组

示例

            // 哪个日期最接近2015-8-6
            const dateToCompare = new Date(2015, 8, 6);
            const datesArray = [
              new Date(2015, 0, 1),
              new Date(2016, 0, 1),
              new Date(2017, 0, 1)
            ]
           const result = closestIndexTo(dateToCompare, datesArray)
           //=> 1
