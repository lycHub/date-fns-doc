closestIndexTo

与给定日期相比，从数组中返回最接近日期的索引。 类型 说明 Number 从数组中返回最接近日期的索引。 类型 说明 TypeError 第二个参数必须是数组 
            // 哪个日期最接近2015-8-6，返回其索引
            const dateToCompare = new Date(2015, 8, 6);
            const datesArray = [
              new Date(2015, 0, 1),
              new Date(2016, 0, 1),
              new Date(2017, 0, 1)
            ]
           const result = closestIndexTo(dateToCompare, datesArray)
           //=> 1
