closestTo

与给定日期相比，从数组中返回最接近的日期。 类型 说明 Date 从数组中返回最接近的日期。 类型 说明 TypeError 第二个参数必须是数组 
            // 返回最接近2015-9-6的日期
            const dateToCompare = new Date(2015, 8, 6);
            var result = closestTo(dateToCompare, [
              new Date(2000, 0, 1),
              new Date(2030, 0, 1)
            ])
            //=> Tue Jan 01 2030 00:00:00
