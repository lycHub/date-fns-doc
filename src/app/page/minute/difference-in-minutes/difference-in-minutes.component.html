differenceInMinutes

获取给定日期之间的分钟数。 类型 说明 Number 分钟数。 类型 说明 TypeError 必须要两个参数 
           // 2 July 2014 12:07:59 和 2 July 2014 12:20:00 相差多少分钟?
           var result = differenceInMinutes(
             new Date(2014, 6, 2, 12, 20, 0),
             new Date(2014, 6, 2, 12, 7, 59)
           )
           //=> 12
